**SQL commands** are extensively used to **interact with databases**, enabling users to perform a wide range of actions on database systems
**SQL Commands are mainly categorized into five categories:**
- **DDL** (Data Definition Language): Define the database schema
	CREATE, DROP, ALTER, TRUNCATE
- **DQL** (Data Query Language): Performing queries on the data within schema objects
	SELECT 
- **DML** (Data Manipulation Language): manipulation of data present in the database
	INSERT, UPDATE, DELETE
- **DCL** (Data Control Language): Deal with the rights, permissions, and other controls of the database system
	GRANT, REVOKE
- **TCL** (Transaction Control Language): Control the change version of databases
	BEGIN TRANSACTION, COMMIT, ROLLBACK
